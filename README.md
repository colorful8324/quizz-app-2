# quizz-app-2
Chức năng có trong API:
- Câu hỏi (tạm thời chưa có phân quyền)
+ Tìm 1/tất-cả câu hỏi
+ Thêm câu hỏi
+ Cập nhật câu hỏi
+ Xóa câu hỏi

- Đề thi
+ Tạo đề thi ngẫu nhiên với số câu hỏi, loại câu hỏi tùy chỉnh
+ Xem đề thi
+ Chấm điểm bài thi

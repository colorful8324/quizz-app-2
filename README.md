# quizz-app-2
Chức năng có trong API:
- Câu hỏi (tạm thời chưa có phân quyền)
_ Tìm 1/tất-cả câu hỏi
_ Thêm câu hỏi
_ Cập nhật câu hỏi
_ Xóa câu hỏi

- Đề thi
_ Tạo đề thi ngẫu nhiên với số câu hỏi, loại câu hỏi tùy chỉnh
_ Xem đề thi
_ Chấm điểm bài thi
